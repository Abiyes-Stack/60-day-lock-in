# pipeline/

Empty on purpose. `/lock-in-setup` populates this directory for **your** goal:

- `tracker.csv` — your source of truth, schema generated for your unit of work
- Subfolders + `_TEMPLATE.md` files for each artifact your generated pipeline produces
- This README gets overwritten with your weekly flow, status definitions, and column reference

To see what a fully populated pipeline looks like, open `../examples/job-search/pipeline/`.

**Privacy:** once populated, this directory contains your real pursuit. Make your fork private, or uncomment the privacy section in the root `.gitignore`.
